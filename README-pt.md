## :clipboard: Sobre

Este projeto foi desenvolvido seguindo o vídeo [AUTENTICAÇÃO COM GITHUB](https://www.youtube.com/watch?v=OE4IupjQjzE), do Lucas Nhimi no Youtube, que tem como objetivo realizar autenticação em projetos Next.js usando o NextAuth.

### Tecnologias utilizadas

- [Next.js](https://nextjs.org/)
- [NextAuth.js](https://next-auth.js.org/)
- [Typescript](https://www.typescriptlang.org/)
- [Prisma](https://www.prisma.io/)
- [PlanetScale](https://planetscale.com/)
- [Tailwind](https://tailwindcss.com/)
